## Tecnologías Utilizadas

- **React**: Se utilizó React para construir la interfaz de usuario de manera eficiente y dinámica.
- **HTML y CSS**: Se emplearon HTML y CSS para estructurar y diseñar la página.
- **JavaScript**: Se utilizó JavaScript para implementar la lógica del lado del cliente.
- **Firebase Studio**: Esta herramienta tecnológica facilita tener todo el entorno de desarrollo de forma online, permitiendo una gestión eficiente de los recursos y la implementación de funcionalidades en tiempo real.
- **Herramientas de IA**: Se usaron herramientas de inteligencia artificial para generar imágenes, código y este mismo README.

## Instalación

Para clonar y ejecutar este proyecto en tu máquina local, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/mijaliv/CV-Digital-Mijal.git
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd CV-Digital-Mijal
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

4. Inicia la aplicación:
   ```bash
   npm start
   ```
